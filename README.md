# Cistunko.rs

Marketing site for the Cistunko deep-cleaning service, built with Next.js.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm test
npm run lint
npm run build
```

## Launch checklist

- Replace placeholder phone number in `src/lib/site-config.ts`
- Replace placeholder WhatsApp number in `src/lib/site-config.ts`
- Add final proof images in `public/images/proof/`
- Add production OG image in `public/images/og-default.jpg`
- Connect GitHub repo and Vercel project
